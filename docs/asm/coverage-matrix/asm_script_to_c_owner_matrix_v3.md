## Evidence Rules
- exact_name: normalized basename match between scripts/*.asm and src/game/*_scripts.c basename.
- explicit_scripts_ref: C script file contains explicit scripts/<AsmScript>.asm reference.

- event_map_symbol_prefix: event_data.c contains kNpcs_<MapName...> symbol starting with ASM script basename (e.g., Route11Gate -> Route11Gate1F/2F).
- No behavior inference was used.
